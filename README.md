# 🍽️ When2Eat — 美食轉盤推薦平臺

一個幫助選擇困難症的我們,快速決定「今天要吃什麼」的可愛小工具✨  
透過直覺的操作介面與輕鬆的互動體驗,只要轉動轉盤,就能幫你選出下一餐的美味去處!

---

## 🧭 專案簡介

**When2Eat** 是一款以「美食推薦」為主題的輕量級網頁應用程式,目標使用者為開發團隊內部小組成員。  
使用者可透過:
- 🍀 **轉盤功能**: 隨機抽選餐廳
- 🍔 **篩選功能**: 依照餐廳分類、價位等 篩選候選清單
- 🌟 **收藏功能**: 收藏喜愛的餐廳,並在轉盤中增加權重,提高中選機率

> 📌 本專案著重於前端互動與 UI/UX 設計,不開放一般校內使用。

## 🗂️ 功能說明 (HTML/JS 版本)

本專案目前為使用 HTML, CSS, 和原生 JavaScript 打造的單頁式應用程式 (SPA)。主要功能如下：

* ** 多頁面導覽**
  * 應用程式包含「首頁 (`Home`)」、「篩選器 (`Filters`)」、「我的最愛 (`Favorites`)」、「餐廳列表 (`List`)」和「關於我們 (`About`)」等多個視圖。
  * 透過頂部導覽列在不同功能區塊間進行切換。

* **🍔 餐廳篩選 (`Filters`)**
  * 使用者可根據「餐廳類型」、「價格範圍」和「校門區域」等多重條件來篩選候選餐廳。
  * 若篩選後無符合條件的餐廳,系統會彈出提示,避免進入無結果的抽籤。

* **🍀 隨機抽籤 (`Decide Now`)**
  * 根據篩選結果或全體餐廳,執行一個吃義大利麵的可愛動畫,模擬抽籤過程。
  * 動畫結束後,展示抽出的主要推薦餐廳與數個備選餐廳。
  * 結果以輪播卡片 (Carousel) 的形式呈現,可左右切換查看。

* **❤️ 我的最愛 (`Favorites`)**
  * 在餐廳卡片上點擊心形按鈕,可將該餐廳加入或移出收藏。
  * 收藏列表會獨立顯示在 "Favorites" 頁面,並利用瀏覽器的 `localStorage` 進行儲存,重新整理頁面也不會遺失。
  * 在收藏列表中,每家餐廳旁都附有功能按鈕:
    *   **查看詳情 (👁️)**: 點擊後會彈出視窗顯示餐廳基本資訊 (為未來串接 Google Maps 預留位置)。
    *   **刪除 (🗑️)**: 可將該餐廳從收藏列表中移除。

* **🍽️ 餐廳列表 (`Restaurant List`)**
  * 以橫向捲動的方式,展示目前資料庫中所有的餐廳卡片,方便使用者快速瀏覽。
  * **新增即時篩選功能**: 點擊右上角的篩選圖示可開啟篩選面板，根據類型、價格、區域即時過濾餐廳列表。點擊清除圖示可重置篩選。
