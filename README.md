# 🍽️ When2Eat — 美食轉盤推薦平臺

一個幫助選擇困難症的我們，快速決定「今天要吃什麼」的可愛小工具✨  
透過直覺的操作介面與輕鬆的互動體驗，只要轉動轉盤，就能幫你選出下一餐的美味去處！

---

## 🧭 專案簡介

**When2Eat** 是一款以「美食推薦」為主題的輕量級網頁應用程式，目標使用者為開發團隊內部小組成員。  
使用者可透過：
- 🍀 **轉盤功能**：隨機抽選餐廳
- 🍔 **篩選功能**：依照餐廳分類（如中式、西式、甜點…）篩選候選清單
- 🌟 **收藏功能**：收藏喜愛的餐廳，並在轉盤中增加權重，提高中選機率

> 📌 本專案著重於前端互動與 UI/UX 設計，不開放一般校內使用。

---

## 🗂️ 功能說明

- 🎯 **轉盤**  
  隨機抽選餐廳，並在使用者收藏的情況下，調整中選權重。  

- 🍽️ **餐廳分類篩選**  
  依照不同餐廳類型（如早午餐、甜點、火鍋…）篩選候選清單。  

- ❤️ **收藏清單**  
  收藏喜愛餐廳，並儲存在瀏覽器的 LocalStorage 中。  
  ⚠️ 若使用者重新整理或換裝置，資料不會跨設備保存。  

- 🗺️ **Google Map API（規劃中）**  
  - 顯示抽到餐廳的地點資訊  
  - 可與使用者的 Google 帳號串接餐廳清單（進階功能）

## 🚀 專案開發流程

1. **介面設計**：  
   使用 Figma 設計頁面版型 → 確立介面、配色與互動邏輯。

2. **前端開發**：  
   - 建立基本頁面結構（HTML / CSS）  
   - 使用 JavaScript 實作轉盤、收藏與篩選功能

3. **（選擇性）後端規劃**：  
   若需擴充登入系統與資料庫，預留 Django 架構。

4. **部署**：  
   - 網站靜態檔案上傳至 AWS S3  
   - 設定公開存取，提供小組成員使用
